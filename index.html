Joonas Query Language - JQL

V 0.9
NOTE: This app is an early Work-In-Progress!
In the current version, tables can be created, rows can be added to them and the contents of the tables can be viewed.
Many of the features of JQL are still yet to be implemented but some of the functionality is already in place,
mainly: creation of tables and adding data to tables.

To delete all the tables for the current session (removal of tables is not saved but if you make changes after removing the tables, those changes are saved), execute the query:
DELETEALL

Please enter your query below. Click "Execute" to execute it.
