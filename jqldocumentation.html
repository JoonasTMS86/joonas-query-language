JQL Documentation

Joonas Query Language, or JQL for short, is a very straightforward database query language to use if you are familiar with SQL.

JQL is a language that I devised.

JQL is basically identical to SQL in terms of functionality. The syntax in my language is just different.

Overview

SQL has commands such as:

SELECT, INSERT, UPDATE, DELETE, CREATE TABLE


My JQL language has commands which are equivalent to those commands, but in JQL, the respective commands are:

GET, ADD, CHANGE, REMOVE, NEW TABLE.

Viewing the Rows of the Table

The GET command is used for retrieving rows from one or several tables.
The JQL equivalent for SQL "*" (all columns) is the ALL keyword. Eg. the JQL equivalent for SQL "SELECT * FROM tablename" is:

GET ALL tablename

To retrieve specific columns from the table - for example column1, column2, column3 - simply specify the column names the same way as in SQL:

GET column1, column2, column3 tablename

Viewing the Names of All the Tables in the Database

To see what tables exist in the database, simply execute the GET command without any parameters:

GET

Inserting a Row to the Table

The ADD command is used for adding a new row to a table.
Example for a table named "products" which has columns "id", "name", "date", "qty":

ADD products (1, "product1", "2026-03-01", 500)

To add several rows in one query, separate each row with a comma (you might also want to add a line break between each line for better readability but that is not mandatory):

ADD products (1, "product1", "2026-03-01", 500),
(2, "product2", "2026-03-02", 250)

Modifying a Row of the Table

The CHANGE command is used for changing one or several rows of a table.

Care should be taken when using this command if the intention is to modify only one of the rows of the table.
You might want to ensure that you only target the desired row(s) by using a GET query first.

Your table most likely has a unique identifier for each row of the table. "id" for example, which is usually an integer value that automatically increments by one every time a new row is added to the table.
To change a specific row, for example a row with id of value 2, use:

CHANGE products (name "Mouse", qty 300) IF id = 2

The above query changes the "name" column of the requested row ("id" = 2) to "Mouse" and the "qty" column to 300.
If you wish to modify all the rows of your table, for example you wish to name all products "Mouse" and their quantities to 300, just omit the IF condition:

CHANGE products (name "Mouse", qty 300)

Obviously, great care should be taken when executing a command that modifies all records.

Deleting a Row from the Table

The REMOVE command is used for removing one or several rows from a table.

Care should be taken when using this command if the intention is to remove only one of the rows of the table.
You might want to ensure that you only target the desired row(s) by using a GET query first.

To remove the row from the table which has "id" of value 2, use the following query:

REMOVE products IF id = 2

If you wish to delete all the rows of your table but you wish to keep the table itself (so that your table is empty but ready to receive new records), just omit the IF condition:

REMOVE products

Obviously, great care should be taken when executing a command that deletes all records.
